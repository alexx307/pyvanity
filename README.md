### ✨ Features

- 🔍 **Vanity Address Generation**
  - Prefix (start)
  - Suffix (end)
  - Prefix + Suffix combined

- ⚡ **Multi-core CPU optimization**
  - Uses all available CPU cores
  - Auto-configuration based on your hardware

- 🗄️ **Integrated Database (SQLite)**
  - Automatically saves generated wallets
  - Built-in database viewer
  - Double-click to load a wallet

- 🔎 **Advanced Search**
  - Address
  - Private key
  - Mnemonic
  - ID

- ⚙️ **Auto Hardware Detection**
  - CPU (cores / threads)
  - RAM
  - GPU (info only)
